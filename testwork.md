Тестовое задание: парсер сайта
Задача
Реализовать на Python парсер сайта одного домена.
Парсер принимает стартовый URL, обходит страницы сайта и возвращает контактные данные, найденные на сайте.

Входной параметр
start_url: str
Абсолютный URL сайта.

Выходные данные
Результат работы — объект следующего формата:
{
	"url": str,
	"emails": list[str],
	"phones": list[str]
}
Описание полей
url — основной URL сайта
emails — найденные e-mail адреса
phones — найденные телефонные номера
Если данные не найдены, возвращаются пустые списки.

Требования
Парсер должен работать с реальными сайтами
Обходить только страницы одного домена
Искать контактные данные не только на главной странице
Корректно обрабатывать относительные и абсолютные ссылки
Быть устойчивым к ошибкам сети и HTML-разметки
Иметь ограничения, предотвращающие бесконечный обход сайта
Разрешено использовать любые библиотеки Python

Ограничения
Нельзя хардкодить сайты или домены
Нельзя рассчитывать на валидный HTML

Что будет оцениваться
Делает ли решение ровно то, что требуется
Думает ли кандидат о граничных ситуациях
Насколько предсказуемо и аккуратно работает код
Читаемость и логичность реализации

Ожидаемый результат
Парсер вызывается одним вызовом (передали URL, получили результат)
Возвращаемый объект соответствует описанному формату
Решение можно проверить на произвольном сайте без доработок
